
Developer Instructions:

Follow the instructions to run the app:
1. Clone https://github.com/acharaakshit/COV-AID.git to your system.
2. Open the project in Android Studio.
3. Connect the project to a Firebase Database, details of which have been provided below.
4. In Android Studio, go Build->Build APKS and Bundle to build the corressponding APK file.
5. Locate the APK file and copy it into your android device to install and run the app.

Note: To run this project, you will also need to generate your [google maps api key](https://developers.google.com/maps/documentation/android-sdk/get-api-key).
      After generating the key, go to app->res->values->google_maps_api.xml and put your own api key. 
      	

Follow the instructions to connect the app to Firebase:

1. Go to https://console.firebase.google.com
2. Login with the account you wish to connect the database to.
3. Make a new project and follow the instructions provided after that.
4. After creating the project, go to project settings page and download the latest google-services.json file.
5. Copy and paste the downloaded file to 'COV_AID/app' folder. 
6. Use the Firebase console website or Firebase Assistant provided in Android Studio to verify if the project has been connected to the database.
